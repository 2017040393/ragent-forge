# Related Projects

> 语言: [English](related_projects.md) | 中文

RAGentForge 受到更广泛的 RAG、agent 和 evaluation 生态启发，包括：

- LangChain
- LangGraph
- LlamaIndex
- Haystack
- Dify
- RAGFlow
- AnythingLLM
- Ragas
- DeepEval

这些项目探索的是更宽泛的 frameworks、orchestration、no-code workflows、
enterprise knowledge bases 和 evaluation systems。RAGentForge 早期聚焦更窄：
保持小、本地、可检查、TUI-first，并以学习为导向。

目标不是替代这些工具。目标是给开发者一个清晰的本地 workbench，用来理解自己
的文档如何流经 Agentic RAG workflow。
